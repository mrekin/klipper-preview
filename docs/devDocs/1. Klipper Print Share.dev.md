# 1. Klipper Print Share - Доработка

## Общее описание задачи

Реализация сервиса Klipper Print Share для генерации временных ссылок на просмотр статуса 3D-печати. Операторы могут создавать ссылки с ограниченным временем жизни (TTL) и опциональными комментариями, а заказчики — просматривать прогресс печати, температуры, слои и визуализацию G-code без необходимости регистрации.

Согласно RSR был выбран **Вариант B (модифицированный)**: исправление критических ошибок, добавление комментариев к токенам, реализация QR-кодов, исправление интеграции с Moonraker API и сохранение настроек.
